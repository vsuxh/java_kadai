ファイルの読込み

読み込みたいファイルの名前を与えてFileReaderクラスのインスタンスを生成し, それをBufferedReaderクラスのインスタンスで包む(HAS-A関係になる).

その後, BufferedReaderクラスのreadLineメソッドで, ファイルの内容を1行ずつ読み込んでいく. readLineメソッドはファイルの最後に到達するとnullを返すので, 戻り値がnullになったらループを終了する.

最後にcloseメソッドでファイルを閉じる.

※FileReaderクラスはファイルからの1文字単位での読込みを担当し, BufferedReaderクラスはそれをバッファリングして1行ごとにまとめて読み込む, というように役割分担がなされている.

なお, try & catchによる例外処理機構については実行中のエラー処理の定石として覚え, 以下の例を真似て利用してみて欲しい. tryブロック中で実行時エラー(ファイルが無い, ファイルが壊れている, アクセス権限が無い等)が発生すると, catchブロックが実行される仕組みになっている.

※皆さんが普段多く悩んでいる(かもしれない)コンパイル時のエラーではないので注意.

import java.io.*; // 要インポート文追加

// ----ファイルを読み込みたいメソッドの中で----
try
{                               // ↓ここにファイル名を与える.
    FileReader fr = new FileReader("test.txt"); // ファイルを開く.
    BufferedReader br = new BufferedReader(fr);
    String buf;
    while ((buf = br.readLine()) != null) // 1行読み込み, それがnullでなければ,
    {
        System.out.println(buf); // bufを処理する(この例では表示するだけ).
    }
    br.close();
    fr.close(); // ファイルを閉じる.

    // closeを個別に2回呼び出す事の必要性について, 世の中に様々な流儀があるが,
    // 個別に2回呼び出しても問題ないことが確認されているため, ここでは個別2回流を採用する.
}
catch (IOException ex) // もしファイル入出力でエラーが起きたら,
{
    System.out.println("ファイル読込みエラー");
    System.exit(1); // 実行終了
}

ファイルの書込み

基本的な構成は読込みと同様である. FileWriterクラスのインスタンスをBufferedWriterクラスのインスタンスで包み, writeメソッドでファイルに文字列を書き込む. 最後にcloseメソッドでファイルを閉じる. エラー処理の方法も読込みと同様である.

                            // ↓ここにファイル名を与える.
FileWriter fw = new FileWriter("test.txt"); // ファイルを開く.
BufferedWriter bw = new BufferedWriter(fw);
bw.write("お急ぎですか?");                  // ファイルに書き込む.
bw.close();
fw.close();                                 // ファイルを閉じる.

この様に, Javaでのファイル入出力は, バッファとなる配列の用意やポインタ操作, バッファオーバーフローへの配慮が不要なため, C言語でのfopen, fgets等を用いたファイル入出力よりも, 扱いが容易かつ安全である.

また, その他にも画像ファイル(GIF, JPEG, PNG等)の入出力や, URL指定によるネットワーク経由でのファイル読込み, インスタンスのファイルへの保存と復元(シリアライズ機能)など, C言語と比べて近代的な機能が標準ライブラリで多数用意されている.

ウェブサーバーからの読込み(GET)

上記のFileReaderクラスはJavaのプログラムが動作しているコンピューター内にあるローカルファイルの読込みであったが, HttpURLConnectionクラスとInputStreamReaderクラスを用いると, ウェブサーバー上のファイルをGETして読み込むことができる. これにより, ウェブサーバーから情報を入手して自作アプリに表示することも可能になる. 以下にプログラムの例を示す.

import java.net.*; // 要インポート文追加

// ----try節の中で----
               // ↓ここにアドレス(URL)を与える.
URL url = new URL("http://jsuri1.info.kushiro-ct.ac.jp/~tenmo/java/hello.txt");

HttpURLConnection http = (HttpURLConnection)url.openConnection();
http.setRequestMethod("GET");
http.connect();

InputStreamReader isr = new InputStreamReader(http.getInputStream());
BufferedReader br = new BufferedReader(isr);
