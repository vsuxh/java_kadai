プログラミング言語IIIA(Java) テーマ06

内容

  • メソッド(関数)の作り方・呼び出し方を復習する.
  • 基本的なクラスとコンストラクタの使い方を復習する.

課題

以下の課題のレポートは, レポートファイルreport06.txtを作成してアップロードにより提出すること. なお, 今回のレポートに記載が必要な事項は, 全てプログラムのソースコードのみである.

  1. コマンドライン引数から三角形の高さと底辺の長さを実数で入力し, 面積を出力するプログラムを作成せよ. 以下の条件を満たす様に作成すること.
    • ソースファイル名: Kadai01.java ※今回はソースファイル一つで作成する.
    • クラス名: Kadai01
    • 面積を計算するメソッドのメソッド名: calcArea
    • calcAreaメソッドの引数: double型二つ
    • calcAreaメソッドの戻り値: double型
    • staticメソッドとして作成すること. フィールドなし. コンストラクタなし.
      •     class Kadai01
    {
        static 戻り値の型 calcArea(引数の型 引数の変数名, 引数の型 引数の変数名)
        {
            ...    ↓この部分の「型」を上の「戻り値の型」の所に書く.
            return ...;
        }
        public static void main(String[] args)
        {
            double arg0 = Double.parseDouble(args[0]);
            double arg1 = Double.parseDouble(args[1]);
            ...
        }
    }

      $ javac Kadai01.java
$ java Kadai01 3.0 4.0
    • 標準出力に出力するメソッドにはSystem.out.println(〜)を利用すること. ※以下同様.
    • calcAreaメソッドの中では面積を出力しないこと.
    • main側でcalcAreaメソッドの戻り値をdouble型のローカル変数で受け取ってから出力すること.
    • args[0]などのString型をdouble型に変換するメソッドにはDouble.parseDouble(〜)を利用すること. ※以下同様.
  2. コマンドライン引数から5科目のテストの点数を整数で入力し, その平均点(実数)を出力するプログラムを作成せよ. 以下の条件を満たす様に作成すること.
    • ソースファイル名: Kadai02.java ※ソースファイル一つで作成する.
    • クラス名: Kadai02
    • 平均点を計算するメソッドのメソッド名: calcAverage
    • calcAverageメソッドの引数: int型5つ
    • calcAverageメソッドの戻り値: double型
    • staticメソッドとして作成すること. フィールドなし. コンストラクタなし.
    • calcAverageメソッドの中では平均点を出力しないこと.
    • main側でcalcAverageメソッドの戻り値をdouble型のローカル変数で受け取ってから出力すること.
    • args[0]などのString型をint型に変換するメソッドにはInteger.parseInt(〜)を利用すること. ※以下同様.
  3. コマンドライン引数から整数を1つ入力し, その値に応じて次の様な値を出力するプログラムを作成せよ. 以下の条件を満たす様に作成すること.
    • 値が偶数だった場合0
    • 値が奇数だった場合1
    • ソースファイル名: Kadai03.java ※ソースファイル一つで作成する.
    • クラス名: Kadai03
    • 偶数か奇数かを判定して0か1を返すメソッドのメソッド名: isOdd ※イズオッド
    • isOddメソッドの引数: int型1つ
    • isOddメソッドの戻り値: int型
    • staticメソッドとして作成すること. フィールドなし. コンストラクタなし.
    • isOddメソッドの中では判定結果を出力しないこと.
    • main側でisOddメソッドの戻り値をint型のローカル変数で受け取ってから出力すること.
  4. コマンドライン引数から整数を入力し, その値に応じて次の様な値を出力するプログラムを作成せよ. 以下の条件を満たす様に作成すること.
    • 値が18未満の場合1
    • それ以外の場合0
    • ソースファイル名: Kadai04.java ※ソースファイル一つで作成する.
    • クラス名: Kadai04
    • 18未満の値かどうかを判定して0か1を返すメソッドのメソッド名: isUnder18
    • isUnder18メソッドの引数: int型1つ
    • isUnder18メソッドの戻り値: int型
    • staticメソッドとして作成すること. フィールドなし. コンストラクタなし.
    • isUnder18メソッドの中では判定結果を出力しないこと.
    • main側でisUnder18メソッドの戻り値をint型のローカル変数で受け取ってから出力すること.
  5. 以下の仕様を満たす, 整数値の座標を表す MyPoint クラスを作成せよ. (main ではない)
    • ファイル名: MyPoint.java
    • クラス名: MyPoint
    • フィールド: int 型で x と y と z
    • コンストラクタ: int 型の x と y と z を引数で受け取り, それぞれフィールドの x と y と z に代入する
    • メソッド: メソッド名 print, 引数無し, 戻り値無し, フィールドの x と y と z の値を標準出力へ出力する
    • ※この課題では static はどこにも付けない.
  6. 以下の仕様を満たす, 課題5の MyPoint クラスを用いる main メソッドのみを含む MyPointApp クラスを作成せよ.
    • ファイル名: MyPointApp.java
    • クラス名: MyPointApp
    • メソッド: public static void main(String[] args)
    • コマンドライン引数から三つの整数を入力する(自動的に args[0] と args[1] と args[2] に入ってくる)
    • args[0] を整数型に変換し, ローカル変数 x に代入する
    • args[1] を整数型に変換し, ローカル変数 y に代入する
    • args[2] を整数型に変換し, ローカル変数 z に代入する
    • オブジェクト変数 mp を宣言する
    • ローカル変数 x と y と z を MyPoint クラスのコンストラクタへ渡し, インスタンスのアドレスを mp に代入する
    • mp に対して print メソッドを呼び出す
